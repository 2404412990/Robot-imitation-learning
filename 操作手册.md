### 支持的机器人

| 机器人 | DOF | Agent 脚本 | Robot Key |
|--------|-----|-----------|-----------|
| Unitree G1 | 29 | G1mimicAgent.cs | `unitree_g1` |
| Unitree H1 | 19 | H1mimicAgent.cs | `unitree_h1` |
| X02Lite | 10 | X02LiteMimicAgent.cs | `x02lite` |

### 工作模式

| 模式 | 说明 |
|------|------|
| **回放 (Replay)** | 从 CSV 文件播放预录制动作，无物理（Kinematic 传送） |
| **实时 (Live)** | 从 WHAM+GMR Python 管道实时接收动作数据 |
| **训练 (Training)** | PPO 强化学习，PD 控制器跟踪参考姿态 |

### 数据流
```
视频/摄像头 → WHAM (人体姿态估计) + GMR (运动重定向)
    → live_motion.csv
    → StartInput 协程轮询 → Agent.FixedUpdate
    → ArticulationBody PD 驱动关节
```

| 软件 | 用途 | 下载 |

> VS 2019 安装时必须勾选 **"使用 C++ 的桌面开发"**。

```powershell

# 3. 等待包自动安装（首次打开 ~10 分钟）
```

```powershell

conda activate gewu
pip3 install torch~=2.2.1 --index-url https://download.pytorch.org/whl/cu121
pip install -r requirements-gewu.txt   # 位于 Assets/Imitation/
mlagents-learn --help                   # 验证
```

### 2.4 Python `wham_gmr` 环境（WHAM+GMR 管道）

**前置条件**: 安装 CUDA 11.3 + VS 2019 Community

```powershell
# 在 Anaconda PowerShell Prompt 中
cd Robot-imitation-learning/           # 位于 Assets/Imitation/Robot-imitation-learning/
.\setup_env.ps1                         # 自动安装脚本
```

> PyTorch3D 和 DPVO 是可选组件，编译失败可跳过，不影响核心功能。

手动安装步骤参考: `Robot-imitation-learning/docs/INSTALL_WIN.md`

## 4. X02Lite 机器人适配

### 4.1 X02Lite 规格

| 属性 | 值 |
|------|-----|
| DOF | 10（纯下肢，每腿 5 个关节） |
| 关节 | L_hip_yaw, L_hip_roll, L_hip_pitch, L_knee_pitch, L_ankle_pitch, R_hip_yaw, R_hip_roll, R_hip_pitch, R_knee_pitch, R_ankle_pitch |
| CSV 格式 | 17 列 (3 根位置 + 4 根旋转 + 10 DOF) |
| Robot Key | `x02lite` |

### 4.2 已创建的文件

| 文件 | 说明 |
|------|------|
| `Robot-imitation-learning/.../ik_configs/smplx_to_x02lite.json` | GMR IK 重定向配置 |
| `X02LiteMimicAgent.cs` | Unity Agent 脚本，实现 IMimicAgent |
| `dataset/x02lite/neutral_stand.csv` | 示例数据集 (200 帧站立姿态) |

### 4.3 已修改的文件

| 文件 | 修改内容 |
|------|---------|
| `Scripts/StartInput.cs` | 添加 `x02lite` 到 SupportedRobotNames |
| `Scripts/StartInput.cs` | 添加别名 `X02Lite`/`X02` |
| `Scripts/StartInput.cs` | 添加 `PopulateRoboListDropdown()` 自动填充 |
| `Scripts/StartInput.cs` | 修正 bashWorkingDirectory 默认路径 |

### 4.4 场景配置（在 Unity Editor 中完成）

1. **附加 Agent 脚本**：选中 X02Lite GameObject → Add Component → `X02LiteMimicAgent`
2. **Behavior Parameters**：`Vector Observation Space Size = 25`，`Continuous Actions = 10`
3. **Scene Robots 映射**：StartInput 的 `sceneRobots` 添加 `Label = "X02Lite"`，`Robot Root = X02Lite GameObject`
4. **保存场景**

### 4.5 添加新机器人的步骤（供参考）

1. 准备 URDF/XML 文件 → `Assets/urdf/<robot>/`
2. 在 `params.py` 中添加条目（XML路径、IK配置、根身体名、相机距离）
3. 创建 `smplx_to_<robot>.json` IK 配置
4. 创建 `<Robot>MimicAgent.cs`（复制 G1mimicAgent.cs，修改 DOF/CSV列数/RobotKey）
5. 修改 `StartInput.cs`：`SupportedRobotNames` + `RobotAliases`
6. 创建示例 CSV 数据集
7. 在 Unity 场景中配置

## 5. 运行项目

### 5.1 回放模式（播放 CSV 动作）

1. Unity Editor 打开 `G1.unity` 场景
2. 点击 ▶ Play
3. 在 RoboList 下拉选择机器人 (G1/H1/X02Lite)
4. 点击 **Replay** 按钮，从下拉选动作文件
5. 机器人开始播放 CSV 中记录的动作

### 5.2 实时模式（WHAM+GMR 管道）

**方法 A — Unity UI 启动:**
1. Unity 运行 G1.unity
2. 选择机器人
3. 勾选 **Live** 模式
4. 点击 **Start Input** → 自动启动 Python 管道

**方法 B — 命令行启动:**
```powershell
conda activate wham_gmr
cd Robot-imitation-learning/

# 从视频驱动 H1 机器人
$env:ROBOT='unitree_h1'
$env:RECORD_GMRVIDEO=1
$env:RECORD_WHAMVIDEO=1
$env:VIDEO='docs/video/kunkundance.mp4'
.\run.ps1
```

### 5.3 训练模式（PPO 强化学习）

```powershell
conda activate gewu
cd Assets/Imitation/

# 开始训练
mlagents-learn config.yaml --run-id=<实验名称> --force

# 续训练
mlagents-learn config.yaml --run-id=<实验名称> --resume

# 查看训练曲线
tensorboard --logdir results --port 6006
# 浏览器打开 http://localhost:6006/
```

Unity 侧：
1. 在 Inspector 中勾选机器人的 `Train` 选项
2. 出现 `[INFO] Listening on ...` 后点击 Unity ▶ Play

### 5.4 环境变量说明

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `VIDEO` | `examples/IMG_9732.mov` | 输入视频，`0` = 摄像头 |
| `ROBOT` | `unitree_g1` | 目标机器人类型 |
| `OUTPUT_ROOT` | `output/stream_demo` | 输出目录 |
| `WHAM_USE_AMP` | `0` | 半精度推理 |
| `GMR_TORCH_DEVICE` | `cpu` | GMR 设备 |
| `RECORD_WHAMVIDEO` | `1` | 保存 WHAM 可视化 |
| `RECORD_GMRVIDEO` | `1` | 渲染 MuJoCo 窗口 |

## 6. 常见问题

### 6.1 编译错误

| 错误 | 解决 |
|------|------|
| CS0101 重复类定义 | 检查是否残留 `unity_UI/` 旧脚本 |
| CS0246 命名空间未找到 | 检查 `Packages/manifest.json` 包是否安装 |
| "Missing (Mono Script)" | Inspector 中 Remove + 重新 Add 脚本组件 |

### 6.2 运行时错误

| 错误 | 解决 |
|------|------|
| 下拉列表空 | RoboList 自动填充，检查 SupportedRobotNames |
| 管道找不到路径 | StartInput 的 `bashWorkingDirectory` 设为 `Robot-imitation-learning/` |
| 机器人悬浮/跳跃 | 确认 `Replay = true`，`Train = false` |
| Observations 数量不匹配 | BehaviorParameters 设置正确维度 |

### 6.3 管道错误

| 错误 | 解决 |
|------|------|
| `AssertionError: Faild to load video` | 视频文件不存在，指定有效的 `VIDEO` 路径 |
| `FileNotFoundError: smplx_to_x02lite.json` | 确认 IK 配置文件已创建 |
| PyTorch3D 编译失败 | 可跳过，不影响 CSV 输出 |
| DPVO 编译失败 | 可跳过，不影响局部坐标系重定向 |
